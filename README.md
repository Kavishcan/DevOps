# DevOps Practice

This repository serves as a personal knowledge base and practice log for DevOps concepts, tools, and cloud platforms. It contains daily tasks, notes, and guides, primarily focusing on AWS and general DevOps practices.

## Repository Structure

The content is organized into the following categories:

### ☁️ KodeCloudAWS
Practice tasks related to Amazon Web Services (AWS), focusing on infrastructure setup and management.

- **Day 1**: [Create Key Pair](KodeCloudAWS/Day1_CreateKeyPair.md) - Setting up SSH access keys.
- **Day 2**: [Create Security Group](KodeCloudAWS/Day2_CreateSecurityGroup.md) - Configuring firewall rules.
- **Day 3**: [Create Subnet](KodeCloudAWS/Day3_CreateSubnet.md) - Network segmentation.
- **Day 4**: [Enable Versioning for S3 Bucket](KodeCloudAWS/Day4_EnableVersioningforS3Bucket.md) - S3 object versioning.

### 🐧 KodeCloudDevOps
General DevOps tasks, including Linux system administration and security.

- **Day 1**: [Linux User Setup with Non-Interactive Shell](KodeCloudDevOps/Day1_LinuxUserSetupwithNon-InteractiveShell.md) - User management.
- **Day 2**: [Temporary User Setup with Expiry](KodeCloudDevOps/Day2_TemporaryUserSetupwithExpiry.md) - Managing user account expiration.
- **Day 3**: [Secure Root SSH Access](KodeCloudDevOps/Day3_SecureRootSSHAccess.md) - Hardening SSH configuration.

## Goals
